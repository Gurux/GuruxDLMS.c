## 4. Обработка ошибок (`hal_errors.h`)

Все функции `int` возвращают 0 (`HAL_SUCCESS`) в случае успеха и отрицательное значение в случае ошибки.

- `HAL_SUCCESS` (0): Операция выполнена успешно.
- `HAL_ERROR_GENERIC` (-1): Общая (неспецифическая) ошибка.
- `HAL_ERROR_INVALID_PARAM` (-2): Передан неверный параметр.
- `HAL_ERROR_NO_MEMORY` (-3): Ошибка выделения памяти.
- `HAL_ERROR_TIMEOUT` (-4): Таймаут операции.
- `HAL_ERROR_NOT_SUPPORTED` (-5): Операция не поддерживается.
- `HAL_ERROR_BUSY` (-6): Ресурс занят.
